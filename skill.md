# 野球革命 Rebas - 數據權重參數 (Coefficient) 頁面分析

## 1. 頁面核心功能與用途 (Purpose)
該頁面的主要功能是提供專為**中華職棒 (CPBL)** 環境校準的「進階數據權重參數 (Coefficients)」。因為不同聯盟（如 MLB 與 CPBL）與不同年份的比賽環境（例如曾出現的「彈力球」賽季）差異極大，該頁面計算並公佈了符合台灣職棒現況的基底參數。
核心數據包含：
*   **wOBA 權重與 FIP 常數**：各項上壘結果的加權值與投手獨立防禦率校正值。
*   **得分期望值矩陣 (Run Expectancy Matrix, RE24)**：在特定出局數與壘包情況下，球隊平均能獲得的預期分數。
*   這些數據主要用於進階棒球數據分析、球迷戰術討論（如短打或盜壘的期望值差異評估），以及作為其他進階指標（如 WE 勝利期望值、LI 關鍵指數）的運算基礎。

## 2. 使用者介面佈局 (UI Layout)
頁面採用以**數據為核心的響應式簡潔佈局 (Data-centric Layout)**。整體版面分為上下或左右區塊：
*   **頂部/側邊控制區**：提供主要的操作過濾器（如下拉選單與頁籤）。
*   **主視覺資料區**：以大型、易讀的數據表格（Data Tables）和矩陣（Matrices）為主，去除多餘的裝飾，確保使用者能快速查閱數字。

## 3. 互動元素與操作方式 (Interactive Elements)
為了應對不同維度的數據查詢，頁面整合了以下互動元素：
*   **賽季下拉選單 (Season Dropdown/Filter)**：棒球參數會隨每年的整體環境變動，使用者可透過下拉選單（或年份按鈕）選擇特定的賽季（如 2023、2024 或是歷史賽季），頁面會即時連動更新為該年度的專屬參數。
*   **數據類型頁籤/按鈕 (Tabs/Toggles)**：允許使用者在不同的數據模型間切換，例如從「得分期望值 (Run Expectancy)」切換至「線性權重 (Linear Weights / wOBA)」或「勝利期望值相關參數」。
*   **狀態提示 (Tooltips / Status Indicators)**：部分進行中的賽季可能會標註「數據穩定中」或「動態更新中」等視覺提示，幫助使用者了解當下參數的參考價值。

## 4. 數據、圖表與表格呈現方式 (Data Presentation)
數據的視覺呈現針對棒球統計學的慣例進行了優化：
*   **得分期望值矩陣 (3x8 RE Matrix)**：以標準的二維網格表呈現。列 (Rows) 為 **0、1、2 出局**，欄 (Columns) 為 **8 種壘包狀態**（空壘、一壘有人、一二壘有人...至滿壘）。交會處的儲存格顯示精確到小數點後 3 位的預期得分數字（例如 0.481）。
*   **線性權重對照表 (Weights List/Table)**：以直列或多欄表格清晰列出 wOBA 的各項加權係數（包含 wBB 保送、wHBP 觸身、w1B 一壘安打、wHR 全壘打等），以及 wOBA Scale、FIP Constant (cFIP)、R/W (Run per Win) 等常數。
*   **情境對比**：針對戰術效益的呈現，可能會運用顏色或高對比度的排版，讓使用者直觀感受不同情境（如推進壘包卻增加出局數）對期望值產生的正負向變化。
